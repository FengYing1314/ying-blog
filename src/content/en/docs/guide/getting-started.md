# Slowly Tidying This Place Up

Recently I keep feeling that some things still deserve a place to stay.

Not because they have to be complete, and not because I need to prove anything. Some thoughts, records, and problems I have met are easy to lose when they stay scattered everywhere.

## Start Small

I do not plan to fill this place up from the beginning.

If I can write down what I still remember now, that is already enough. When I come back later, I can at least know why I thought that way and why I made those choices.

:::: card-grid
::: card Add slowly
Add a little when I remember a little.
:::

::: card No rush
Not every note needs to be long.
:::

::: card Keep it
If I can find it later, it already helps.
:::

## A Small Entrance for Myself

Most of the time, I write things not because they must be useful to others, but because I may need them again later.

If you happen to read this, I hope these scattered notes can be a small reference. If not, that is fine too. Just treat it as passing by.
