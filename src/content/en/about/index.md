# About

The `main` branch of Ying Blog is the project site, positioned between a blog, documentation system, and project showcase. It records the implemented static-site foundation, development rules, content boundaries, and project archive.

## Content Scope

The site currently contains four content types: Posts, Docs, Projects, and About. Posts are for project implementation notes, Docs are for project development notes and maintenance rules, and Projects are configuration-driven cards that do not generate detail pages.

## Maintenance Principles

Chinese and English are both first-class languages. New content must ship in both languages and keep slug, categories, tags, and SEO intent aligned. Personal profile data, personal about content, personal taxonomy, and personal articles belong on the `fy-blog` branch and do not go directly into `main`.
