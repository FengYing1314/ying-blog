# 最近想把这里慢慢整理一下

最近总觉得，有些东西还是应该找个地方好好放着。

不是为了写得多完整，也不是为了证明什么。只是有些想法、记录和踩过的坑，如果一直散在不同地方，过一段时间就很难再找回来。

## 先从一点点开始

我不打算一开始就把这里填得很满。

能先把最近还记得的事情写下来，就已经很好了。以后回头看，至少还能知道当时为什么这么想、为什么这么做。

## 也给自己留个入口

很多时候，我写东西不是因为它一定对别人有用，而是因为我自己以后可能会需要。

如果你刚好也看到这里，希望这些零散记录能给你一点参考。要是没有，也没关系，就当是路过看了一眼。
