# 支持说明

## 问题咨询

如果问题直接涉及本仓库的代码、文档、内容与分支规则、静态输出或部署行为，请使用 issue templates。一般使用咨询不进入 issues。

## Bug

使用 bug report issue template，并提供：

- 受影响的路由、命令或文件。
- 复现步骤。
- 预期行为和实际行为。
- 相关时附上 `npm run check`、`npm run test` 或 `npm run build` 的输出。

## 安全问题

不要在公开 issue 中报告漏洞。请遵循 `SECURITY.md`。

## 范围

本仓库是 Vite+、Vue、TypeScript、SSG 和 Markdown 静态站基础。CMS、评论、分析、搜索、草稿、生成社交图和项目详情页不属于当前支持范围。项目规则见 `docs/rules.md`。
